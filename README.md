## ✨ Descripción General
Bienvenido a **Edu Things - Podcast**, un espacio digital donde se comparten ideas, debates y documentales con un enfoque crítico y una mirada que busca enriquecer la conversación. Este proyecto es la plataforma oficial para explorar los episodios de Eduardo Cabrera, ofreciendo una experiencia de usuario fluida y accesible para descubrir contenido relevante y estimulante.

Aquí encontrarás reflexiones personales, análisis profundos y discusiones sobre temas de interés, presentados de una manera que invita a la reflexión y al diálogo.

## 🚀 Características Principales
*   **Exploración de Episodios:** Navega fácilmente por una colección de episodios de video podcast.
*   **Búsqueda Inteligente:** Encuentra episodios específicos utilizando la barra de búsqueda que filtra por título, descripción, categoría y etiquetas.
*   **Filtrado por Categoría:** Organiza y visualiza el contenido por categorías como "contenido" y "debates".
*   **Reproducción Integrada:** Reproduce videos directamente desde YouTube en un modal interactivo, con soporte para modo paisaje en dispositivos móviles.
*   **Diseño Responsivo:** Disfruta de una experiencia óptima en cualquier dispositivo, desde ordenadores de escritorio hasta smartphones.
*   **Accesibilidad Mejorada:** Navegación por teclado y gestión de foco para una mejor experiencia de usuario.
*   **Sistema de Apoyo:** Opción para "invitar a un café" al creador a través de un modal de donación integrado con PayPal.
*   **Cache Busting:** Mecanismos para asegurar que los visitantes siempre vean la versión más reciente del sitio, evitando problemas de caché.
*   **Deep Linking:** Comparte enlaces directos a episodios específicos (`?id=X`) o secciones (`#acerca`).
*   **Meta Tags SEO/Social:** Optimizado con Open Graph y Twitter Cards para una mejor visualización en redes sociales.

## 🛠️ Tecnologías Utilizadas
Este proyecto está construido con tecnologías web estándar, priorizando la ligereza y el rendimiento:
*   **HTML5:** Estructura semántica del contenido.
*   **CSS3:** Estilos personalizados y diseño responsivo.
*   **JavaScript (Vanilla JS):** Lógica interactiva, filtrado, búsqueda y gestión de modales.
*   **GitHub Pages:** Alojamiento y despliegue del sitio web.
*   **YouTube API (Embed):** Para la reproducción de videos.
*   **Google Fonts:** Tipografías personalizadas (`Cormorant Garamond` y `Outfit`).
*   **SVG Icons:** Iconografía ligera y escalable.
*   **PayPal.me:** Integración para donaciones.

## 💡 Cómo Usar
Simplemente visita la URL del proyecto: https://podcast-edu.github.io/
*   **Navegar:** Utiliza el menú superior para ir a "Episodios" o "Acerca".
*   **Buscar:** Escribe en la barra de búsqueda para encontrar contenido.
*   **Filtrar:** Haz clic en los botones de categoría para ver episodios específicos.
*   **Reproducir:** Haz clic en cualquier tarjeta de episodio para abrir el reproductor de video.
*   **Apoyar:** Si te gusta el contenido, puedes hacer clic en el botón "Apoyar" para invitar a un café al creador.
*   **Compartir:** La URL se actualiza automáticamente al abrir un episodio, permitiéndote compartir enlaces directos.
*   **Recargar:** Si recargas la página con un episodio abierto o en la sección "Acerca", el sitio intentará mantener esa vista.

## 🤝 Contribuciones
¡Las contribuciones son bienvenidas! Si encuentras un error, tienes una sugerencia de mejora o quieres añadir una nueva característica, por favor:
1.  Haz un "fork" del repositorio.
2.  Crea una nueva rama (`git checkout -b feature/nueva-caracteristica`).
3.  Realiza tus cambios y asegúrate de que el código sea limpio y funcional.
4.  Haz un "commit" de tus cambios (`git commit -m 'feat: Añade nueva característica'`).
5.  Sube tus cambios (`git push origin feature/nueva-caracteristica`).
6.  Abre un "Pull Request" detallando tus modificaciones.
